 CalculadoraJava

Proyecto de ejemplo: calculadora simple en Java.

## Características
- Operaciones: Suma, Resta, Multiplicación, División.
- Manejo de errores (división entre cero).
- Interfaz por consola.
- Código modular: `Calculadora` y `Main`.

## Cómo ejecutar
1. Clonar el repositorio:
```bash
git clone https://github.com/tuusuario/CalculadoraJava.git

    Compilar:

cd CalculadoraJava/src/main/java
javac com/ejemplo/calculadora/*.java

    Ejecutar:

java com.ejemplo.calculadora.Main
